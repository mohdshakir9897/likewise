== Description ==

By using this plugin you can create your own Filter Portfolio Gallery.

[FREE VERSION DEMO LINK](http://filterportfoliogalleryfree.phoeniixxdemo.com/gallery-images/) | [FREE VERSION DOCUMENTATION LINK](http://filterportfoliogalleryfree.phoeniixxdemo.com/wp-content/uploads/2018/04/Filter-portfolio-gallery-free-doc.pdf)

= Overview =

This is the very simple plugin by which one can create a beautiful portfolio gallery and can upload the images in the gallery.

= Free Features: =

* Option to add the category from the backend settings.
* Option to upload the images for a particular category.
* Option to edit or delete any category from the backend settings.
* Can shuffle the images in ascending or descending order on the basis of position or description of the images.
* Option to search for a particular image.
* Option to upload multiple images at a time.

== Installation ==

1. Install [WordPress](https://wordpress.org/)
2. Go to your WordPress admin panel, and then to Plugins. Click on "Add New" and then upload the zip file of the plugin using the "Upload Plugin" button you can find on top of the screen.
